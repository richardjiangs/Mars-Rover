# Ares 机械减震与保温系统补充说明


### 步骤 1：搭建“摇臂-转向架”底盘与机械减震系统

本项目的“机械减震”不采用汽车弹簧减震器，而采用火星车常见的 `Rocker-Bogie / 摇臂-转向架`。它的核心不是“弹簧变软”，而是让每侧 3 个轮子通过两个活动铰链上下摆动，使轮子遇到障碍时车身不会被直接顶起。

#### 1.1 一侧三轮的结构

每一侧由一根长摇臂和一根小转向架组成：

```text
一侧侧视图：

                 中央车身盒 / 主控舱
             ┌────────────────────┐
             │ ESP32 / 电池 / 驱动 │
             └─────────O──────────┘
                       │ 主铰链 A
                       │
前轮电机+轮子  ○────────┘────────O  二级铰链 B
                                  │
                           中轮 ○─┴─○ 后轮
                              小转向架 Bogie
```

拆开看就是：

```text
长摇臂 Rocker：
前轮电机位 ───── 主铰链 A ───── 二级铰链 B

小转向架 Bogie：
中轮电机位 ───── 二级铰链 B ───── 后轮电机位
```

- 前轮压到障碍物时，长摇臂绕主铰链 A 转动。
- 中轮或后轮压到障碍物时，小转向架绕二级铰链 B 转动。
- 这样可以让六个轮子尽量保持接地，把冲击分散到多个轮子和连杆上。

#### 1.2 推荐尺寸（配 65mm 左右 TT 越野轮）

| 零件 | 建议尺寸 | 说明 |
| --- | --- | --- |
| 中央车身盒 | 约 100–160mm 长 | 放电池、ESP32、L298N、LM2596 |
| 每侧长摇臂 | 约 220–260mm | 前端固定前轮，后端连接小转向架 |
| 每侧小转向架 | 约 140–170mm | 两端固定中轮、后轮，中间接长摇臂 |
| 主铰链 A | M4/M5/M6 螺丝 + 垫片 | 必须能转动，不能粘死 |
| 二级铰链 B | M4/M5 螺丝 + 垫片 | 必须能转动，不能粘死 |

孔位示意：

```text
长摇臂：
前轮电机位        主铰链 A              二级铰链 B
   O──────────────O────────────────────O

小转向架：
中轮电机位        二级铰链 B            后轮电机位
   O──────────────O────────────────────O
```

#### 1.3 制作步骤

1. 先做中央车身盒：用 ABS 接线盒、塑料盒、木板盒或亚克力盒作为主控舱，两侧各加一块侧板，用来安装主铰链 A。
2. 做左右两根长摇臂：材料可用 10x10mm 铝方管、木条、PVC 扁条、亚克力条或金属尺。左右必须尽量等长。
3. 做左右两个小转向架：中间打孔接二级铰链 B，两端分别固定中轮和后轮电机。
4. 铰链处用“螺丝 + 垫片 + 螺母”，不要用热熔胶粘死。拧紧后稍微回松，让摇臂能自由摆动但不要明显晃动。
5. 每个 TT 电机最好先固定在小电机座或小木片上，再用扎带穿孔固定到摇臂末端，最后少量热熔胶防滑。不要只靠一大坨热熔胶承重。
6. 在摇臂可能撞到车身的位置贴海绵、EVA 或泡沫块作为限位缓冲。也可以用橡皮筋轻轻拉住摇臂，让它回到中间角度，但橡皮筋不要作为主要承重件。
7. 电机线必须留出弯折余量，避免摇臂上下摆动时把线拉断。

#### 1.4 第一版简化方案

如果制作难度太高，可以先做“简化 Rocker-Bogie”：

- 不做每个轮子的独立转向；
- 六个轮子方向固定；
- 用左右两侧差速 / 坦克转向完成转弯；
- 先用便宜的 `1:48 TT 电机` 验证结构和接线，成功后再统一换成 `1:90` 或 `1:120`。

注意：六个驱动电机必须尽量使用同一减速比，不要左右混用不同减速比，否则会明显跑偏。



### 步骤 3.5：制作保温与温度保护系统（-20°~50°目标）

本项目的温度目标应分成两层理解：

- **模型演示层面**：用保温盒、温度传感器和程序保护逻辑，展示火星车具备“低温休眠、高温保护”的工程设计。
- **真实工程层面**：普通 TT 电机、18650 电池、ESP32 模块并不等于真的可以长期在 -20°C 到 50°C 环境中满负荷工作。因此本项目不建议使用没有温控的电阻加热片或把锂电池放进极端环境里测试。

#### 3.5.1 被动保温盒结构

把电池、ESP32、ESP32-CAM、传感器接口尽量放进中央主控舱内，做成“夹层保温盒”：

```text
外部环境
  │
  ▼
┌────────────────────┐
│ 外壳：ABS盒/塑料盒   │  防撞、防灰尘
│ ┌────────────────┐ │
│ │ 保温层：EVA/海绵 │ │  减少热量流失
│ │ ┌────────────┐ │ │
│ │ │ 空气层/泡沫 │ │ │  形成隔热缓冲
│ │ │ ESP32/电池  │ │ │  核心电子舱
│ │ └────────────┘ │ │
│ └────────────────┘ │
└────────────────────┘
```

可用材料：EVA 泡棉、薄海绵、泡沫板、瓦楞纸板、热缩管、绝缘胶带、少量铝箔胶带或急救毯反光膜。铝箔只能贴在外层或夹层，不能接触电路板焊点和电池极片，防止短路。

#### 3.5.2 冷环境策略

- 电池和主控板放在保温盒中心，不要贴在底盘外侧。
- L298N 会发热，不能完全包死；可让散热片靠近通风孔或外壳内壁。
- 低温时先进入 `COLD_START`：上电后原地等待 30–60 秒，让电子舱内部温度稳定，再低速行驶。
- 当温度过低时，火星车进入 `COLD_SLEEP`：关闭电机，只保留主控低功耗监测，等待光照或温度恢复。

#### 3.5.3 热环境策略

- 电子舱内部温度超过 45°C 时降低电机功率，减少 ESP32-CAM 连续图传。
- 温度超过 50°C 时进入 `THERMAL_PAUSE`：停车、停止拍摄、等待降温。
- 外壳顶部可使用浅色材料或反光膜；但太阳能板表面不能被覆盖。

#### 3.5.4 温度传感器选项

建议加一个可选温度传感器，便于在电脑端显示内部温度：

- 简单方案：`DS18B20` 数字温度探头，装在主控舱内部靠近电池的位置。
- 替代方案：`10k NTC` 温度模块，接 ESP32 的 ADC 输入脚。

推荐程序规则：

```text
inside_temp < -10°C  -> COLD_SLEEP，禁止电机运行
-10°C ~ 0°C          -> COLD_START，只允许低速短时运行
0°C ~ 45°C           -> NORMAL，正常运行
45°C ~ 50°C          -> HOT_LIMIT，降低速度和拍摄频率
inside_temp > 50°C   -> THERMAL_PAUSE，停车降温
```

#### 3.5.5 安全边界

- 不要用裸电阻丝、没有温控的加热片直接贴在 18650 电池上。
- 不要让铝箔、金属胶带、螺丝接触电池正负极或电路板焊点。
- 不要把 L298N、LM2596 完全包进厚海绵里运行，防止自身过热。
- 真正低温测试时，不建议把锂电池放进冰箱或冷冻环境反复测试；课堂模型可用冰袋放在盒子外侧进行短时间模拟，并全程人工看护。


## 建议实际制作顺序

1. 先用纸板或木条做一侧 Rocker-Bogie 小模型，只验证两个铰链能否摆动。
2. 再把这一侧复制成左右两侧，固定到中央车身盒。
3. 先不装全部电子元件，只装 6 个轮子和电机，推动车子看六轮是否都能接地。
4. 接 L298N 和 ESP32，先让车架空测试六个轮子的方向。
5. 最后再装保温盒、温度传感器、摄像头和太阳能板。

## 最小可行版本

- 六轮方向固定，不做独立转向。
- V1 使用 1:48 TT 电机验证结构。
- Rocker-Bogie 只做两个能活动的铰链：每侧主铰链 A + 二级铰链 B。
- 保温系统先做被动保温盒，不上主动加热。
- 温度传感器可选；如果没有传感器，可以先用“保温结构说明 + 程序伪代码”满足作业展示。
