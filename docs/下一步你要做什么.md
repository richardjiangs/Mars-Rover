# 下一步你要做什么（用户行动清单）

## ① 3D 打印（可以马上开始）

**直接打印 `Print required/plates/` 里的 6 个整盘**，参数与零件对照表见
`Print required/README.md`。全部平躺免支撑，最大占床 215×240（你的 250 床够用）。
补单件时用 `Print required/parts/`。旧 STL（含 *_ARES_FIXED_V9）已确认有错并删除，
不要再用。

## ② 补采购（部件照片里没有的东西）

必买：
- [ ] **65mm 越野软胎 ×6**（照片里没看到轮子；跌落方案要求空心/海绵大脚软胎，硬胎会折断电机轴）
- [ ] **2S 平衡充电器（8.4V）**（太阳能已取消，这是唯一充电手段）
- [ ] **M3 硅胶减震球 ×4+**（F2 主控板悬浮用，无人机飞控配件几块钱）
- [ ] **M3 尼龙铜柱套装（25/35mm）+ M4/M5 螺栓垫片防松螺母**（楼层支撑与铰链）
- [ ] **电阻：1kΩ×3 + 2kΩ×3**（三路超声波 Echo 分压，必须）
- [ ] **碳纤维细圆棒 2 根（或 PVC 扎带）**（甲虫穹顶）+ 高强度小弹簧或 TPU 件（桅杆底座）
- [ ] EVA 泡棉（含 5mm 高密度）、魔术贴绑带、22AWG 硅胶线、热缩管

可选（建议买，几块钱）：
- [ ] LDR 光敏模块（带 AO/DO）——休眠唤醒演示更真实（装上后 `config.h` 置 `ENABLE_LDR 1`）
- [ ] DS18B20 + 4.7kΩ——温度演示（`ENABLE_DS18B20 1`）
- [ ] 100kΩ + 33kΩ 电阻——电池电压遥测（`ENABLE_VBAT 1`）
- [ ] 红蓝 3D 眼镜——看伪 3D 成果

## ④ 烧录两块板（件到齐前也能先做）

1. 装 Arduino IDE + ESP32 支持 + 2 个库（5 分钟，步骤：`firmware/commander/README.md` 第一节）。
2. 电脑开热点；两块板的 `config.h` 填热点名/密码；CAM 的 `PC_HOST` 填电脑 IP。
3. 烧 Commander（ESP32 Dev Module）→ 串口记下车 IP。
4. 烧 CAM（AI Thinker ESP32-CAM，用下载器底板；**若已接串口线先拔掉**）→ 记下相机 IP。
5. dashboard 填两个 IP，浏览器应看到遥测 + 画面。

## ⑤ 装配（打印件到手后）

按 `docs/装配与测试指南.md` 阶段 ①→④：先单侧摇臂 → 整车结构 → 四层主舱 → 接线（**LM2596 先调 5.00V，Echo 必须分压，全部共地**）。

## ⑥ 标定与逐级测试

按指南 T0→T8 顺序：架空联调 → 5m 直线（调 `YAW_KP`，把实测速度回填 `CM_PER_SEC_CRUISE` 和 plot_map 的 `--speed`）→ 避障 → 25° 坡 → 拍摄 → 休眠 → 建图 → **最后**才做 2m 跌落。

## ⑦ 遇到问题优先查

| 现象 | 先查 |
| --- | --- |
| 某侧车轮方向反 | 对调该侧电机线，或换 `config.h` 该侧 FWD/REV |
| MPU6050 初始化失败 | I2C 接反 → `config.h` 互换 SDA/SCL |
| 超声波 F/L/R 对不上 | `config.h` 对调三组 Trig/Echo 定义 |
| 遥测 cam 一直 OFF | 串口线 25/23 是否接、是否烧录后忘接回 |
| L298N 发烫/车没力 | 降 `MOTOR_MAX_PCT`；考虑换双 L298N（接法在 GPIO Map §1.1） |
| 直线跑偏 | 加大 `YAW_KP`；六电机减速比是否一致 |
