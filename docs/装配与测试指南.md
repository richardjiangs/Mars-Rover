# Ares 装配与测试指南（V3）

> 依据：`docs/楼层布局.md`（四层布局）、`docs/Ares_机械减震与保温系统补充说明.md`（摇臂制作与顺序）、`docs/2米高空坠落生存与自恢复方案.md`（抗坠落件）、`hardware/Ares_V3_GPIO_Map.md`（接线唯一权威）。
> 原则：**每完成一级测试再进入下一步装配**，不要一口气装完再通电。

## 一、装配顺序

### 阶段 ①：结构（无电子件）

1. 3D 打印全部结构件（清单与参数见 `docs/下一步你要做什么.md`）。
2. 先装**一侧** Rocker-Bogie：长摇臂 + 小转向架 + 主铰链 A（Φ5）+ 二级铰链 B（Φ5）。
   螺栓 + 垫片 + 尼龙防松螺母，拧紧后回松 1/4 圈：能自由摆动、不明显旷量。**禁止热熔胶粘死铰链。**
3. 单侧验证通过后装另一侧，两侧务必等长对称。
4. 六个电机装到摆臂末端（螺丝/扎带承重，热熔胶只防滑），装轮子。
5. 摆臂与主舱可能磕碰处贴 EVA 限位缓冲；电机线预留摆动余量。
6. **结构验收**：整车放平推动，六轮全程着地；单轮压 20mm 高的垫块，其余五轮不离地。

### 阶段 ②：主舱四层（断电装配）

按 `docs/楼层布局.md`：

1. **B1**：底板铺 5mm 高密度 EVA → 电池盒居中 → 两道魔术贴勒紧（不打螺丝）。
2. **F1**：两块 L298N 贴左右边缘（若单板方案则居中靠边）、LM2596 放中缝；用 35mm 尼龙铜柱撑起上层（散热片上方留 ≥8mm）。
3. **F2**：80×60 洞洞板（ESP32 MAIN + MPU6050 焊死、排针刚性固定、MPU 水平）；四角用 **M3 硅胶减震球**连接，不用硬铜柱。
4. **天台**：柱子（视觉桅杆）过 TPU/弹簧底座装 SG90 + ESP32-CAM；碳纤维棒交叉扎成 X 穹顶（最高点须高于摄像头）；钮子开关**横装**在尾部侧板 + U 型护翼（护翼比拨杆长 ≥5mm）。
5. **配重**（50%+ 填充打印件）装在底部指定位，压低重心。

### 阶段 ③：接线（按 GPIO Map，先电源后信号）

1. 电池 → 2S BMS → 总开关 → 7.4V 总线；LM2596 调出 **5.00V**（万用表，空载与带载各测一次）再接任何板子。
2. 接 L298N（7.4V）与六电机（左组 OUT1/2、右组 OUT3/4）。
3. 接信号线：电机 IN1-4→GPIO16/17/18/19；三路超声波（**每路 Echo 经 1k/2k 分压**）；I2C（<10cm）；舵机 GPIO4；UART2（25/23 ↔ CAM 3/1）。
4. 检查单：所有 GND 共地 ✓ / Echo 都有分压 ✓ / 电机线成对绞合、比直线距离长 20% ✓ / 信号线远离电机线 ✓ / L298N 散热片不被泡棉包死 ✓。

### 阶段 ④：软件

两块板的烧录步骤见 `firmware/commander/README.md` 与 `firmware/cam/README.md`；电脑端见 `vision/README.md`。

## 二、T0–T8 分级测试（每级通过才进下一级）

| 级 | 名称 | 做法 | 通过标准 | 覆盖目标 |
| --- | --- | --- | --- | --- |
| T0 | 桌面单件 | 面包板上逐件点亮：ESP32 烧录、MPU 读数、单路超声波、舵机扫动、CAM 出图 | 每件工作正常 | — |
| T1 | 架空联调 | **车轮离地**上电，dashboard 连接：WASD 前后左右、E-STOP、AUTO 挥手触发避障动作 | 方向全对；E-STOP 立即锁死；L298N 连转 5 分钟不烫手（>60℃ 需换双板） | 1 |
| T2 | 直线与速度标定 | 地面 5m 直线跑 3 次；卷尺量横向偏移；秒表测速度 | 偏移 ≤ 30cm（调 `YAW_KP/KD` 后复测）；把实测 cm/s 回填 `CM_PER_SEC_CRUISE` | 2 |
| T3 | 避障 | AUTO 模式，前方 1m 放纸箱，左右不对称留口 | 距箱 ~20cm 停车，转向空旷侧通过，10 次 ≥8 次成功 | 5 |
| T4 | 爬坡 | 木板逐步垫高 10°→15°→20°→25°（防滑面） | 25° 能匀速爬完 1m 不溜坡不翻车（不行→换 1:90/1:120 电机或加大轮纹） | 3 |
| T5 | 拍摄任务 | dashboard 点全景/伪 3D；电脑跑 receive_server | 全景 7 张全收到且能拼接；伪 3D 两张视差明显、能出红蓝图 | 10.1/10.2 |
| T6 | 休眠唤醒 | 演示按钮或遮光入睡；亮光/定时唤醒 | 入睡后电机断电；唤醒后 wake_count +1；80 次触发冲刺模式（可临时把 `WAKE_LIMIT` 改小验证） | 8 |
| T7 | 建图 | 2×3m 围出区域，AUTO 巡航 3 分钟 + telemetry_logger 录制 | plot_map 出图，轨迹与实际路线拓扑一致、障碍点贴近边界 | 10.3 |
| T8 | 跌落（最后做！） | 检查全部防护件到位（软胎/EVA/减震球/穹顶/桅杆弹性/开关护翼）→ 30cm→1m→2m 逐级跌落到**草地或缓冲垫** | 每级落地后遥测正常、可继续行驶；倒扣时穹顶+低重心能翻正（必要时 MPU 检测翻转后轮子助翻） | 4、11 |

安全：T1 前必须架空；T4/T8 旁边随时准备按 E-STOP；T8 全程人员离开落点，逐级进行，任何一级失败先修复再继续，**不要直接上 2m**。

## 三、温度演示（-20°~50° 目标的模型化验收）

1. 装保温盒（EVA 夹层，L298N/LM2596 留散热通道）。
2. 若装了 DS18B20（`ENABLE_DS18B20 1`）：冰袋贴盒外侧 → 遥测 `thermal` 依次出现 COLD_START/COLD_SLEEP；吹风机远距离加热 → HOT_LIMIT/THERMAL_PAUSE，随后自动恢复 NORMAL。全程人工看护，**锂电池不进冰箱**。
3. 未装传感器：以保温结构 + `firmware/commander/thermal.h` 的状态机代码作为设计演示。
