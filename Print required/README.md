## 轮子与电机怎么装（不会掉、不磨臂）

轮子是 84×45、12mm 内六角轮毂、中心孔 Φ4；电机是 1:48 黄色单轴 TT（体 70×22×18，
官方数据）。传动与保持链路：

1. **电机 → 摆臂**：电机体贴摆臂内侧面，2 颗 **M3×30** 顺轴向贯穿 摆臂(外侧已开
   3mm 沉槽)+电机体，尾端 M3 螺母锁死。腰形槽 ±4mm 兜孔位公差；轴从 Φ8 孔穿出。
2. **转接头 → 电机轴**：转接头鼻端伸进摆臂外侧 Φ24 沉台（不接触，留 0.5mm），
   DD 孔（Φ5.6/平面距 3.9）套上 TT 双 D 轴（轴与孔的平面传扭，不打滑）；
   侧面 M3×8 顶丝（配槽内 M3 螺母）顶在轴平面上 → **转接头不会轴向脱出**。
3. **轮子 → 转接头**：轮毂 12mm 内六角套上转接头六角（11.85 对边，微紧配合，
   太紧用锉/砂纸轻修，太松包一层薄胶带）；**M4×20 从轮心 Φ4 孔穿入，锁进转接头
   内的 M4 螺母**（螺母从侧槽滑入），把轮毂压在六角台肩上 → **轮子不会掉**。
4. 转动检查：轮内侧面距摆臂外面 3.5mm（已校验），全程无摩擦；六轮共面着地。

## 装配顺序

见 `docs/装配与测试指南.md`。要点：铰链 A/B 拧紧后回松 1/4 圈保持转动，**严禁热熔胶粘死**；
Rocker 与舱壁之间放 M5 PTFE 垫片，Rocker 与 Bogie 之间放 M4 PTFE 垫片。
