# PROJECT ARES V2.2 - 核心物料采购清单 (BOM)

设计预算 Target: $100.00 USD

## 1. 大脑与控制中枢 (指挥与导航)

- [ ] 1x ESP32 NodeMCU 开发板 (ESP-WROOM-32)  
  主指挥官。必须带 Wi-Fi 和蓝牙，引脚尽量多（通常是 38 针版本）。
- [ ] 1x ESP32-CAM 模块 (带 OV2640 摄像头排线)  
  3D 扫描仪与导航员。用于拍摄 360° 全景和传回电脑端处理。
- [ ] （可选）ESP32-CAM 下载器底板 (约 10 元)  
  强烈建议购买！ESP32-CAM 本身没有 USB 接口，带底板方便你用 USB 线烧录代码。

## 2. 能源与太阳能系统 (电力核心)

- [ ] 2x 18650 动力锂电池  
  建议购买大厂的正品电池（如 2500mAh 甚至 3000mAh），确保续航。
- [ ] 1x 18650 双节串联电池盒 (带引线)  
  注意：必须是串联（输出 7.4V），不能买并联（3.7V）的！
- [ ] 1x 2串 (2S) 锂电池 BMS 保护板  
  核心安全件。7.4V (8.4V满电) 充电必备，防止太阳能板过充导致电池起火。
- [ ] 1x LM2596 DC-DC 降压模块  
  必须带稳压调节旋钮。将 7.4V 电池电压稳稳降到 5.0V，保护主板不被烧毁。
- [ ] 3x 5V 微型太阳能滴胶板  
  尺寸推荐约 130mm x 150mm（几瓦功率即可）。一块固定在顶端，两块作为折叠翼。
- [ ] 1x 船型开关 / 拨动开关 (大电流)  
  安装在电池和降压模块之间，作为整车的总电源开关。

## 3. 动力与底盘行走系统 (六轮驱动)

- [ ] 6x TT 直流减速电机 (V1: 1:48；V2: 1:90；重车/爬坡版可选 1:120，双轴或单轴)  
  建议先用便宜 1:48 验证车架、接线和程序，成功后再统一换成 1:90；不要左右混用不同减速比。
- [ ] 6x 越野 TT 轮胎 (外径约 65mm)  
  表面有较深防滑胎纹的橡胶轮，专为沙地/火星模拟地形设计。
- [ ] 2x L298N 双路电机驱动模块  
  带巨大铝制散热片的那种经典红板。左侧三个电机并联接一块，右侧接另一块。

## 4. 传感器与执行机构 (有效载荷)

- [ ] 1x MPU6050 六轴陀螺仪/加速度计模块  
  姿态传感器。火星车能走“绝对直线”全靠它。
- [ ] 3x HC-SR04 超声波测距传感器  
  前置防撞“雷达”阵列。平价实现激光雷达的避障效果。
- [ ] 3x SG90 微型舵机 (180度版)  
  经典的蓝色小舵机。2个用于展开太阳能板，1个用于带动摄像头云台左右扫射。
- [ ] （可选）1x DS18B20 数字温度传感器 + 4.7kΩ 电阻  
  放在主控舱内部靠近电池的位置，用于显示内部温度，并触发低温休眠 / 高温保护。

## 5. 机械结构与骨架五金 (航天级底盘)

- [ ] 1x 20x20mm 铝合金方管 (壁厚 1-1.5mm，长 1 米)  
  中央脊椎主梁。非常轻但极其坚固。
- [ ] 1x 10x10mm 铝合金方管 (壁厚 1mm，长 1 米)  
  用于切割制作两侧的摇臂-转向架 (Rocker-Bogie) 悬挂臂。
- [ ] 结构挂载舱 (外壳) (约 15 元)  
  如果没有 3D 打印机，买几个 ABS 塑料防水接线盒（Project Box），尺寸约 100x60x40mm，钻孔后挂在铝管上。
- [ ] 螺丝螺母五金套装  
  `M4/M5/M6` 螺栓 + 垫片 + 尼龙防松螺母：用于 Rocker-Bogie 主铰链和二级铰链，必须顺滑且不松动。  
  `M3` 螺丝 + 尼龙/铜柱隔离柱：用于固定所有电路板（防短路）。
- [ ] 橡皮筋 / 小弹力绳 / 海绵块 / EVA 泡棉  
  用于摇臂限位、软缓冲和中央主控舱抗震。橡皮筋只负责回位和限位，不作为主要承重件。

## 6. 保温与热控材料

- [ ] EVA 泡棉 / 薄海绵 / 泡沫板 / 瓦楞纸板  
  用于制作中央电子舱的夹层保温盒，保护 ESP32、ESP32-CAM、18650 电池和传感器接口。
- [ ] 绝缘胶带 / 热缩管 / 扎带  
  用于防止铝箔、线头、焊点造成短路。
- [ ] （可选）铝箔胶带或急救毯反光膜  
  只能贴在外壳外侧或绝缘夹层中，不能接触电池极片或电路板焊点。

## 7. 线材与工具 (基础耗材)

- [ ] 杜邦线套装 (公对公、公对母、母对母各一排)
- [ ] 22 AWG 硅胶软线 (红黑两色，用于走大电流的电源和电机线)
- [ ] 热缩管 / 电工胶布 / 小扎带 (整理走线，防止航天器变成“盘丝洞”)
- [ ] 电烙铁及焊锡 (必须焊接马达引脚和总线)
- [ ] 手电钻 & 阶梯钻头/麻花钻头 (用于在铝合金方管上打孔)
- [ ] 钢锯 (截断铝管使用)

> 提示：在淘宝搜索元件时，直接复制条目中的核心关键词即可找到精准匹配的零件。
